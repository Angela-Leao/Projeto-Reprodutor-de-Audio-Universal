#*Player universal*

As linguagens principais serão: 
Python para análise e pesquisa
C++ para velocidade e processamento de áudio
HTML, css e java script para a interface de usuário 

O app deve ser capaz de extrair e reproduzir o áudio de links da web, youtube, spotify ou áudios extraidos de redes sociais.
O app deve ser capaz de funcionar em segundo plano, com ou sem internet, e com armazenamento em nuvem. A presença de um áudio em primeiro plano não deve interferir no áudio em segundo plano. Deve ter interface interativa e com tutorial opcional.
